# Peter Faller's Blog Space

This is a record of some of the things that I have come across while traversing the internet. It's
largely aimed at being a helper for those 'Now where did I come across that before?' moments.

## Previewing

Serve the site and watch for markup/sass changes

```
$ jekyll serve
```

View your website at http://127.0.0.1:4000/
