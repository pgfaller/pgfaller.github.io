After going around in circles and struggling to get Joplin installed everywhere, an alternative popped up that I didn't see the first time around - and that is, to use the GitHub Wiki for this site's pages project.

The limitations are:
- GitHub markdown rendering does not cater for KaTeX or memrmaid diagrams

The advantages are:
- accessible from anywhere via GitHub
- fully rendered markdown can be viewed using VSCode extensions 'Markdown Math' and 'Markdown Preview Mermaid Support'
