So I tried to use GitHub pages and Jekyll as a means of taking notes;
but it turned out to be cumbersome - remembering to update the index file each time,
remembering to do a commit each time, ... too fiddly.

Thus I have reverted to using [Joplin](https://joplinapp.org/)
and its synchronisation via [Dropbox](https://dropbox.com).

The advantages are:
- synchronisation across systems (although practically limited by me not having a paid Dropbox subscription)
- multi-platform: Windows, Linux, Android, ...
- able to include mathematical formulas using KaTEX
